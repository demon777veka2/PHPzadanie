Cистемных требованиях
  ● ОС рабочего места;
	Windows 10 64-bit
  ● Браузер для запуска;
	Яндекс, mozilla firefox. Google Chrome, Opera.
  ● Программное обеспечение;
	Apache 2.2.8,
	PHP 5.2.6.

Bнструкцией по запуску:
1)Файл PHPzadanie перенесите в серверную папку ~/htdocs.
2)Откройте ваш браузер.
3)В адресной строке введите localhost/PHPzadanie/index.php.
4)Пользуйтесь приложением.
